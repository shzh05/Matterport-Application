<h1>1. Run npm install in root directory</h1>

<h1>2. Download the SDK Bundle</h1>
Download the official SDK bundle package from the link below:
<a href="https://static.matterport.com/showcase-sdk/bundle/26.3.1_webgl-362-g4ce0c71cc2/showcase-bundle.zip">Download Showcase Bundle v26.3.1</a>

<h1>3. Rename and move Showcase Bundle folder</h1>
Rename the showcase bundle folder to "bundle"
Place it under public folder
File structure should look like this:<br>
      my-matterport-app/<br>
      ├── public/<br>                 
      │   └── bundle<br>
      ├── src/<br> 
      │   └── main.js<br>
      │   └── style.css<br>
      ├── node_modules<br>                                
      ├── index.html<br>              
      └── README.md<br>
